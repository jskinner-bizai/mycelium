The four promises

01

Bi-directional

Every limb informs the brain; the brain can move every limb. One bus. No sensor side, no actuation side.

02

Self-defending

Anomalies and failed health-checks fire real events. A watchdog subscribes, investigates, responds — without a human tap.

03

Self-healing

Every long-running process is supervised. Crashes back off and restart. Cloud failures queue locally. The mesh degrades, it doesn't collapse.

04

Self-improving

Agents can write code that extends the organism. New adapter, new reflex, new memory — proposed, reviewed, committed, redeployed. The organism grows.

The minimum viable organism

import { Bus, Supervisor, MqttAdapter } from '@mycelium/core';

const bus = new Bus({
  adapter: new MqttAdapter({ url: 'mqtt://localhost:1883' }),
  logPath: './events.db',
});
await bus.connect();

bus.on('motion:detected', async (evt) => {
  await bus.publish('light:turn-on', { room: evt.room });
});

const sup = new Supervisor();
sup.register({ name: 'frigate', run: async () => { /* camera pipeline */ } });
sup.register({ name: 'brain',   run: async () => { /* llm loop */ } });
await sup.start();

Run it

Two TypeScript packages under Apache-2.0:

  • @mycelium/core — the substrate. Event bus, supervisor, adapter primitives.
  • @mycelium/brain — the reasoning organ. Provider-agnostic — bring any LLM.

Two recipes to start from:

  • starter — minimum-viable organism, no hardware required
  • homelab-nuc — full single-host organism with eyes, brain, autonomic NS, immune system, skin
